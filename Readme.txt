Deskripsi Tugas:
Buatlah website portofolio pribadi menggunakan HTML, CSS, dan JavaScript yang terdiri dari 4 bagian utama, yaitu:
1.Home : Menampilkan sapaan selamat datang, nama Anda, dan deskripsi singkat (tagline atau perkenalan).
2.About : Menampilkan biodata singkat seperti nama lengkap, pendidikan, hobi, dan keterampilan.
3.Portfolio : Menampilkan daftar proyek atau karya Anda (boleh berupa gambar, deskripsi proyek, atau tautan).
4.Contact : Menampilkan form kontak sederhana (nama, email, pesan) dan informasi media sosial Anda.

Spesifikasi Teknis

1.Gunakan HTML untuk struktur konten halaman.
2.Gunakan CSS (boleh internal atau eksternal) untuk mempercantik tampilan website.
3.Gunakan JavaScript minimal untuk satu fitur interaktif, seperti:
a.Validasi form kontak
b.Efek tombol/menu
c.Smooth scroll atau efek sederhana lainnya
d.Navigasi antar bagian dapat dilakukan melalui navbar atau menu.
4.Responsif: tampilan tetap rapi di perangkat desktop dan mobile.

Struktur Folder

2.Rancanglah desain tampilan semenarik mungkin dengan menggunakan HTML, Css dan Javascript.
